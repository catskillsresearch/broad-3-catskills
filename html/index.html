Solution Overview

Our Catskills Solution for Crunch 3 of the Broad Institute IBD Challenge focuses on identifying gene markers that distinguish dysplastic (pre-cancerous) from non-dysplastic tissue regions in inflammatory bowel disease samples. We've developed a comprehensive approach that integrates techniques from previous crunches while adding specific innovations for Crunch 3.

Multi-level Spatial Context

Integrates information from spot, sub-spot, and neighborhood levels to capture both local and global tissue patterns.

Cell-wise Optimization

Optimizes for preserving the relative expression levels of different genes within each cell for more accurate cellular state representation.

Small Dataset Approach

Starts with a small, well-labeled dataset for rapid experimentation and validation before scaling to the full dataset.

Hyperparameter Optimization

Uses Weights & Biases for systematic hyperparameter exploration and experiment tracking.

Streamlined Pipeline

Our solution features a streamlined pipeline that removes unused data elements and clearly shows the transformations from input data to final gene rankings. The pipeline is implemented using Luigi, which manages dependencies between tasks and ensures that each step is executed in the correct order.

We've organized the pipeline to start with a small dataset for rapid experimentation and hyperparameter optimization before scaling to the full dataset. This approach allows for faster iteration and validation of the model architecture and hyperparameters.

Integrated Techniques

Our solution integrates techniques from various approaches to effectively identify gene markers that distinguish dysplastic from non-dysplastic tissue regions:

  • DeepSpot Techniques: Multi-level spatial context integration, deep-set neural network architecture, and pathology foundation models
  • Cell-wise Optimization: Preserving cellular state and function through cell-wise Spearman optimization
  • Log Fold Change Analysis: Identifying differentially expressed genes between dysplastic and non-dysplastic regions
  • Statistical Significance Testing: Ensuring robustness of findings through rigorous statistical analysis

Small Dataset Approach

Our solution starts with a small, well-labeled dataset for rapid experimentation and validation before scaling to the full dataset. This approach allows for:

  • Faster Iteration: Experiments run in minutes instead of hours or days
  • Rapid Hyperparameter Exploration: Test many hyperparameter combinations quickly
  • Early Validation: Verify that the model can learn the task before scaling
  • Debugging: Easier to identify and fix issues with a smaller dataset

We use Weights & Biases for systematic hyperparameter exploration and experiment tracking, allowing us to find the optimal model configuration efficiently.
